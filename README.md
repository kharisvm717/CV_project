# CV_project — ИИ-ассистент для прескоринга кандидатов

Проект по модулю **PE10.7 — ИИ-ассистент. Прескоринг кандидатов на вакансии**.

Приложение на **Streamlit**, которое:
- принимает текст **вакансии** и **резюме**,
- при желании умеет парсить данные по ссылкам с `hh.ru`,
- отправляет запрос в `OpenAI`,
- возвращает текстовый анализ и оценку кандидата.

## Как запустить локально

1. Установите зависимости:

   ```bash
   pip install -r requirements.txt
   ```

2. Создайте файл `.streamlit/secrets.toml` и добавьте в него:

   ```toml
   OPENAI_API_KEY = "ваш_ключ_от_OpenAI"
   ```

3. Запустите приложение:

   ```bash
   streamlit run streamlit_app.py
   ```

Приложение откроется в браузере по адресу `http://localhost:8501`.

## Деплой на Streamlit Community Cloud

1. Загрузите этот проект в репозиторий GitHub.
2. Перейдите на https://streamlit.io/cloud и создайте новое приложение,
   указав ссылку на ваш репозиторий и файл `streamlit_app.py` как главный.
3. В настройках приложения (`Settings → Secrets`) добавьте:

   ```toml
   OPENAI_API_KEY = "ваш_ключ_от_OpenAI"
   ```

После этого приложение будет доступно по публичной ссылке — её можно отправлять на проверку вместе со скриншотами.
